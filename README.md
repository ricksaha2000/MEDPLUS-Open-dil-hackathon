# INSTRUCTIONS
1) Setup Virtual Environment(LINUX)

$ virtualenv venv

$ source venv/bin/activate


2) Install requirements

$ pip install -r requirements.txt


3) Migrate db File

$ python manage.py makemigrations

$ python manage.py migrate


4) Launch server

$ python manage.py runserver
